Chmod 1753

Chmod 1753 (chmod a+rwx,g-w,o-r,ug+s,+t,u-s,g-s) sets permissions so that, (U)ser / owner can read, can write and can execute. (G)roup can read, can't write and can execute. (O)thers can't read, can write and can execute.

Owner Rights (u) Group Rights (g) Others Rights (o)
Read (4)
Write (2)
Execute (1)
Extra chmod command options







Use the octal CHMOD Command:

chmod -R 1753 folder_name
OR use the symbolic CHMOD Command:

chmod -R a+rwx,g-w,o-r,ug+s,+t,u-s,g-s folder_name

Chmod Permissions for chmod 1753

Chmod owner

  • Owner can read
  • Owner can write
  • Owner can execute

Chmod group

  • Group can read
  • Group can't write
  • Group can execute

Chmod other

  • Others can't read
  • Others can write
  • Others can execute

How chmod 1753 looks in file listing

For files

After changing a file's mode to 1753 the file's mode will be displayed in Unix style file lsting as: -rwxr-x-wt

For folders

After changing a directory's mode to 1753 the folder's mode will be displayed in Unix style file lsting as: drwxr-x-wt
